例如

  • 常量：

    • 字符常量可以是中文 例如’中’是一个常量 但是字符常量有且只有一个字符 比如 ‘’ 没有字符 ‘ab’ 这种都是错的

    • 字符串常量可以是 “” 空字符

    • 空常量：null 表示没有任何数据

      System.out.println(null);这种写法是错误的，打印空常量

  • 数据类型分类：

    • 基本数据类型
      • 整数型 byte(1字节) short(2字节) int(4字节)(int默认类型) long(8字节)
      • 浮点型 float(4字节) double(8字节) （double默认类型）
        • 近似值 都不精确
      • 字符型 char**(2字节)** 可表示中文
      • 布尔型 boolean(1字节)
    • 引用数据类型
      • 字符串 引用类型
      • 数组
      • 类
      • 接口
      • Lambda

  • 变量：程序运行期间 内容可以改变

    [数据类型] [变量名称];

  • 变量注意事项：

    • byte num=400; 错误 右方数据范围不能大于左边的范围

    • 对于float和long类型注意字母后缀。

    • 变量使用不能超过作用域的范围 从变量定义开始，然后一直到直接所属的大括号为止

    • 可以多个变量同时定义与赋值

      int a=1,b=2,c=3;

  • 当数据不一样时，将会发生数据类型转换

    • 自动类型转换

      • 数据范围从小到大 （与字节数不一定相关 例如float的范围大于long 但是字节数少）

        long lon=100; //将Int赋给lon

        double dou=1.0f;//将float赋给double

        float flo=30L;//将long赋值给float

    • 强制类型转换

      • 数据范围从大到小

        int num=(int) 100L;

      • 可能会发生精度损失，数据溢出

        int num2=(int)3.5; //输出3 将小数部分直接截取 精度运算

        int num1=(int) 1000000000000000000L 数据溢出

  • byte/short/char类型都可以进行数学运算 在运算的时候，都会首先被提升为int类型，然后再进行运算

  • boolean不能进行任何数据转换

  • 数字和字符的对照表（编码表）

    • ASCII码表：American Standard Code for Information Interchange 美国信息交换标准代码
    • Unicode码表：万国码 开头0-127与ASCII码表一致 后面有各国的的字符 例如中文、日文等

  • / 整除 结果是商 整数

  • % 取模 结果是余数 只有对整数而言才有余数

  • 一旦运算中有不同类型的数据，那么结果将是数据范围大的类型

  • 加号“+”

    • 对于字符char，计算之前会将char提升为int，然后进行运算

    • 对于String类型（不全是小写 所以不是关键字） 加号表示连接字符串。无论什么类型与字符串连接，最后结果都是字符串。

      优先级问题：

      “String”+20+30 结果是”String2030”

  • 自增自减、复合赋值运算符中隐含了一个强制类型转换

  • 与&& 或|| 具有短路效果 如果左边已经可以获得结果 右边将不会执行，从而节省一定的性能
