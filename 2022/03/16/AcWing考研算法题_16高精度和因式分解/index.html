N的阶乘

3389. N 的阶乘 - AcWing题库
观察题目的数值范围，很明显long long类型的变量也不能够存储下，所以需要用string来进行存储，即大数乘法。

下面这种解法虽然正确，但是由于计算量大所以会超时，因此需要优化。

由题目，n的范围是1-1000，而根据阶乘的性质，n的阶乘是n-1的阶乘再乘以n。如果多次输入n并分开计算n的阶乘，每次计算>m的数的阶乘的时候，都需要再次计算m的阶乘，所以考虑一次计算完所有1-1000数据的阶乘并存储到f[N]中，然后求n的阶乘即返回f[n]，这样只需要计算一次1000!即可，而不用重复计算。
#技巧 预处理“打表”
如果输入是多种情况，而这些情况的结果互相关联，比如大数由小数经过某些计算获得，那么可以通过直接计算一遍所给数据最大值的情况，同时将计算过程中的小数对应输出结果进行存储，即构成一个表。那么实际输入获得输出时，只需要查表即可，避免了重复计算而超时。

例题

质因数的个数

3380. 质因数的个数 - AcWing题库
质因数(或质因子)在数论里是指能整除给定正整数的质数。两个没有共同质因子的正整数 称为互质。因为 1 没有质因子,1 与任何正整数(包括 1 本身)都是互质。正整数的因数分 解可将正整数表示为一连串的质因子相乘, 质因子如重复可以指数表示。 根据算术基本定理, 任何正整数皆有独一无二的质因子分解式。只有一个质因子的正整数为质数。
#知识点
质因子分解的时候，n中最多包含一个大于$\sqrt{n}$的质因子。

证明：如果n有两个质因子q和p，那么q*p *其他质因子 > n显然不合理。

同时，如果n没有小于等于$sqrt(n)$的质因子，那么他的质因子是他本身。
结论：1到n中的质数个数大概在$\frac{n}{\ln{n}}$个。
筛选质数的复杂度大概为O(n)
#好题 这个求解质因子的方法很好！

约数

约数，又称因数。整数a除以整数b(b≠0) 除得的商正好是整数而没有余数，我们就说a能被b整除，或b能整除a。a称为b的倍数，b称为a的约数。
3377. 约数的个数 - AcWing题库
解法一
直接遍历1到$\sqrt{n}$的数，看哪些数是n的因数，然后就加2。（因数除了i*i=n以外都是两两对应出现，所以加2）

解法二
#技巧
通过计算获得约数的个数。
已知数n，那么n可以被分解为k个质因子$p_i$的$m_i$次幂的和。
$$
n = p_1^{m_1}p_2^{m_2}…p_k^{m_k}
$$
那么，n的约数个数为：$(m_1+1)(m_2+1)…(m_k+1)$
因此问题转换为求解数n的质因子以及对应的个数。

阶乘末尾的零

3507. 阶乘的末尾0 - AcWing题库
$1 * 2 * … * n = 2^a * 5^b * c$，则末尾零的个数为$\min{a, b}$
对于一个数的阶乘，5这个因子一定是每隔5个数就至少出现一次，即：
n! = 1 * 2 * 3 * 4 * (1 * 5) * ... * (2 * 5) * ... * (3 * 5) * ... * n
即每隔5个数出现一个5，所以计算n/5（下取整）即可得到每隔5个数出现的5的个数。
然后继续观察发现，每隔25出现的是两个 5，所以除了每隔5个数算了一个 5，每隔25个数，还需要多算一个 5。也就是我们需要再加上 n / 25 （下取整）个 5。
同理继续分析53、54、… 即可得到n!的因子5出现的个数：(都是下取整)
$$
n/5 + n / 5^2 + n/ 5^3 + …
$$

#好题 推论：计算阶乘n!中质因子p的个数
$$
num_p = n/p + n/p^2 + n/p^3 + …
$$
如果所求因子为合数c，可以将这个合数c拆分为质数的乘积（即求出其质因子$a_i$和质因子的幂次$b_i$），c的在n!中出现的个数就是
$$
num_c = \min(num_{a_1}/b_1, num_{a_2}/b_2,…, num_{a_i}/b_i)
$$
