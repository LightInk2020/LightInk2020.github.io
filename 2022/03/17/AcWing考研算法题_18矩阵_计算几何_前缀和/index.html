#知识点 memset、memcpy用法
memcpy
memcpy用来做内存拷贝，可以用它拷贝任何数据类型的对象，可以指定拷贝的数据长度。
void *memcpy(void *dest, const void *src, size_t n);
含义：从源src所指向的内存空间地址的起始位置开始拷贝n个字节到目标dest所指的内存地址的起始位置。
memset
memset的作用是对一段内存空间每个字节全部设置为某个值，常用于内存空间的初始化。
void *memset(void *s, int c, size_t n);
含义：把已开辟的内存空间s的前n个字节设置为c。

#知识点
注意：类的定义后面要加分号;
类构造函数可以写成ClassName(int _x, int _y):x(_x), y(_y){}
#知识点 浮点数是否相等的判断
因为浮点数存储有误差存在，所以一般比较时只需要两个数差值绝对值小于ε即可，一般ε取$10^{-8}$。例如比较a和b的差值可以写成：
$$
|a - b | < 10^{-8}
$$
代码：fabs(a - b) < 1e-8

fabs()返回浮点数的绝对值，abs()返回整数的绝对值。

前缀和

最长平衡串

#好题
3553. 最长平衡串 - AcWing题库
分析：用$a_i$和$b_i$分别表示前i个字符中1的个数和0的个数，子串$s_{ij}$表示第i到第j个字符构成的串，如果$s_{ij}$为平衡串，则$a_j - a_{i-1} = b_j - b_{i-1}$，即$a_{i-1} - b_{i-1} = a_j - b_j$，令$t_i = a_i - b_i$，则$s_{ij}$为平衡串等价于$t_{i-1} = t_j$成立。
注意特殊情况，特别是边界情况，例如本题目中m[0] = 0的情况不要忘记！

注意：子串不一定从开头算起，也可能是中间的子串
