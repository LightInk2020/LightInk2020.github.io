数据可视化-Echarts使用

发表于 2022-03-15 分类于 数据可视化 ， Echarts 阅读次数： Valine：
本文字数： 737 阅读时长 ≈ 1 分钟

理论

#知识点
一般后端采用json文件来存储数据，json相当于是js类型文件的子集，所以各种语言多采用json文件存储数据，这样便于前端处理。
#知识点
世界上五大浏览器
google、FireFox、IE、Opera、Safari

html

结构：HTML(内容) + CSS(样式) + JS(动态)
#知识点
脚本语言，不会进行编译，而是直接运行，运行到哪里出了问题就停止在哪里，之前的渲染效果还是起作用。

HTML

head

其中进行声明

  • meta charset 设置编码格式
  • icon 设置页面icon图标
  • title 设置页面标题
  • script 引入script文件

    body

  • id标签 是唯一的

    javascript

    var 声明变量，具体类型由其自动判定
    document 是dom对象，其中getElementById(id)函数可以通过id来获得对应标签。
    js对象，用大括号{}括起来，然后其中都是key:value类型，然后value类型随意。

    矢量图和位图

    矢量图svg

    图片大小变化时清晰度不变
    单个html可以绘制多个图形元素
    高保真

    位图canvas

    图由像素绘制，图片变化时清晰度会变化
    单个html，类似于画笔在画布上画画
    Echarts基于canvas画图

    Echarts

    是一个javascript库，使用时需要引入js文件
    创建一个简单表格的过程：
  • 调用echarts.init(document.getElementById…)来初始化echarts对象
  • 变量option指定图表的数据和配置项
    • title 设置表格标题
    • toolbox 工具箱
    • legend 图例
    • xAxis、yAxis x轴、y轴
    • series[] 是一个数组
      • 其中type指定图标的类型
      • 其中data可以通过js、axios、java、hadoop、mysql等从后端获取
