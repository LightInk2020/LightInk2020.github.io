栈和队列的顺序存储结构

栈

栈顶元素位置（两种情况）：指向最后一个元素、指向最后一个元素的下一个位置

队列

一般采用循环队列。

四种写法 两两组合

一般通过牺牲一个空间避免队满队空判断条件重复

初始化和判定条件，普通队列和双端队列是一样的

队头元素位置：指向第一个元素、指向第一个元素的前一个位置。

队尾元素位置：指向队尾元素、指向队尾元素的下一个位置。

常用-队头指向第一个元素，队尾指向最后一个元素的下一个位置

初始：front = rear = 0

队空：front == rear

队满：front == (rear + 1) % MaxSize

队长：(rear - front + MaxSize) % MaxSize

队头指向第一个元素，队尾指向最后一个元素

初始：front = 0 rear = MaxSize-1

队空：front == (rear + 1) % MaxSize

队满：front == (rear + 2) % MaxSize

队长：(rear - front + 1 + MaxSize) % MaxSize

队头指向第一个元素前一个位置，队尾指向最后一个元素下一个位置

初始：front = MaxSize-1 rear = 0

队空：front == (rear - 1) % MaxSize

队满：front == rear

队长：(rear - front - 1 + MaxSize) % MaxSize

队尾指向第一个元素前一个位置，队尾指向最后一个元素

初始：front = MaxSize-1 rear = MaxSize-1

队空：front == rear

队满：front == (rear + 1) % MaxSize

队长：(rear - front + MaxSize) % MaxSize

栈和队列的应用

栈的应用

C++中函数调用的维护是通过栈来实现的，不过不是所有函数调用都需要，比如尾递归就不需要（不需要维护现场）。

表达式求值（蕴含中缀表达式转后缀表达式、括号匹配）

给定一个表达式，其中运算符仅包含+、-、*、/（加、减、乘、整除），可能包含括号，请你计算表达式的最终值。

题目保证符号-只会作为减号出现，表达式中所有数字均为正整数。

注意题目中整除是向0取整，而不是向下取整，也就是说对于大于 0 的结果向下取整，例如 5/3=15/3=1，对于小于 0 的结果向上取整，例如 5/(1−4)=−1。C++中就是向0取整。
