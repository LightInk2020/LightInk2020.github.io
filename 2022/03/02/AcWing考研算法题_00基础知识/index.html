输入输出

字符char的输入

cin >> c; 不会读入空格
scanf(“%c”, &c); 会读入空格

输入输出

%lf double
%lld long long

printf的输出

%.2f 保留两位小数输出

%

取模运算的结果正负完全取决于被除数。被除数正则除数正，被除数负则除数负。
注意：C++中取模运算本质相当于先对被除数的绝对值和除数的绝对值进行取模运算，然后再加上被除数的符号。
C++取模运算和实际数学中（很多题目中都是要求的数学取余）是不一样的，C++要实现数学的取余运算是**((x%mod)+mod) % mod**。

变量类型转换

对应类型 变量 = (类型)变量;
float转为double 没有问题
double转为float 损失一些精度
int转为浮点数 没问题
浮点数转为int 下取整
int与char之间 ASCII码

强制类型准换

向精度比较高的方向转换。
int向浮点数转换，char向int，int向long long等等

用曼哈顿距离解决菱形输出问题

判断条件特殊用法

cin本身有返回值，例如读文件，如果读到文件末尾EOF会返回0，表示没有读到信息。
scanf本身有返回值，例如读文件，如果读到文件末尾EOF会返回-1，表示没有读到信息。

求数x的所有因子

优化：只需要遍历 i * i <= x，可以得到所有因子，即 i , x/i 的子集。

ASCII码

‘0’ – 48
‘A’ – 65
‘a’ – 97 与大写差值是32
‘\0’ – 0

0 在 int、double、float等中表示0，在char中表示’\0’，在指针中表示NULL，在bool中表示false。

字符串

字符串是字符数组加上一个结束符’\0’

输入输出

char[]常用函数

char[]中的函数，不适用于string

strlen(str) 求字符串的长度，不包含’\0’

strcmp(a, b) 比较两个字符串的大小， a - b 比较的是字典序

一般涉及字符串的大小比较的话，贪心问题

strcpy(a, b) 将字符串b复制给a开始的字符数组

strcat(a, b); 把字符串b拼接到a之后，拼接后的字符串保存在a中

string

算法题目往往用string 而不是char[]

其他类型转为string

C++11增加了全局函数std::to_string，支持将int、long、long long、unsigned、unsigned long、float、double等转为string类型。

string转为其他类型 (实际上是先转为char[])

string转为int – atoi(s.c_str())

string转为float – atof(s.c_str())

string转为long – atol(s.c_str())

string拼接中的一个问题

string拼接，直接用 + // 可以加字符串、字符。

但是必须保证每个加法运算符的两侧的运算对象都至少有一个是string。

sstream用法

更加方便转换string的格式，提取相应信息–特别是原string中有空格的情况。

string的find方法

find()

string中find()返回值是子串在母串中的位置（下标），如果没有找到返回特别的标记npos。（返回值可以看成是一个int型的数）

find_first_of() find_last_of()

find_first_of() 返回子串在母串中首次出现的位置

find_last_of() 返回子串在母串中最后出现的位置

find(子串, index)

查找某个给定位置后(包括这个位置)的子串的位置

查找所有子串在母串中出现的位置
反向查找子串在母串中位置 rfind()

从后面开始找匹配的，比如p=”123as45as”，t=”as”，p.rfind(t)结果是7。通常我们可以这样来使用，当正向查找与反向查找得到的位置不相同说明子串不唯一。

类与结构体

类成员默认是private，结构体成员默认是public

空默认构造函数默认是给出的，如果写了其他构造函数则不会给出，因此最后写其他构造函数时写一下空构造函数。

注意，类与结构体的定义其后面都要加分号。

指针和引用

STL

容器：string, vector, map, stack等

所有容器都有的函数：

vector

基于倍增的思想实现，先开n大小的数组，不够就再开2n大小的数组，然后拷贝。

效率接近普通数组一半。

vector是变长数组，支持随机访问O(1)，不支持在任意位置O(1)插入，为了保证效率，一般在末尾插入。

常用函数

clear
迭代器

迭代器类似STL容器的指针，可以用 * 操作符取到其值

声明示例：

vector<int>::iterator it;

vector迭代器：随机访问迭代器，可以把该迭代器与整数相加减。可以把两个迭代器相减，结果为两个迭代器对应下标之间距离。

begin()/end()

vec.begin(); // 返回指向容器第一个元素的迭代器
vec.end(); // 返回指向容器的最后一个元素的下一个位置的迭代器

遍历
front()/back()

front()返回vector第一个元素的值。back()返回vector最后一个元素的值。

vec.front()等价于vec[0]等价于*vec.begin()

vec.back()等价vec[vec.size-1]

push_back()

向最后插入元素，时间复杂度O(1)

pop_back()

删除最后一个元素，时间复杂度O(1)

insert()

注：O(n)的时间复杂度，一般不用

常用：insert(迭代器, 插入元素) 会将要插入的元素插入到给定迭代器的前面。

元素插入有三种方法。

（1） insert (iterator position, const value_type& val)传入的参数是迭代器的位置和需要插入的元素val。

Position可以是a.begin()，也可以是a.end()，或者这两者中间的一个迭代器位置

（2）void insert (iterator position, size_type n, const value_type& val);

在position位置开始，插入n个值为val的元素

（3）void insert (iterator position, InputIterator first, InputIterator last);

在position位置插入first（比如数组首地址）到last（比如数组首地址+n）之间的元素。

find()

vector没有自带的find()方法。不过可以通过std::find()来实现查找。

比较符号

>、<、==、>=、<=等操作，依次每一个元素比较

技巧

如果返回值类型是vector<int>，而实际返回的数是已知，那么可以直接return {值, 值, ... }

例如返回空的vector，可以 return {};

queue

#include<queue> 头文件中包括循环队列queue和优先队列priority_queue两个容器。

queue 是队列，先进先出

priority_queue 默认是大根堆，优先向外弹出最大值

priority_queue<int> q

priority_queue<int, vector<int>, greater<int>> b; // 小根堆，优先弹出最小值

注意，如果使用priority_queue来存放自己定义的结构体/类，一定要重载小于号（大根堆）/小于号（小根堆）

循环队列queue常用操作

优先队列priority_queue常用操作

stack、queue和priority_queue没有clear()函数

其他容器都有clear()函数

stack

先进后出

常用操作

deque

双端队列，是一个支持在两端高效插入或删除元素的连续线性存储空间。他就像vector和queue的结合，与vector相比，deque在头部增删元素仅需要O(1)的时间；与queue相比，deque支持像数组一样随机访问。

但是实际上其平均操作效率比较低

set

#include<set>包含set和multiset两个容器，分别是“==有序==集合”和“==有序多重==集合”，即前者元素不能重复，而后者可以包含若干个相等的元素。set和multiset内部实现是一棵红黑树，他们支持的函数基本一致。

如果set的元素是结构体/类，注意重载小于号，因为set内部是有序的（默认从小到大），即有比较操作。

常用方法

size/empty/clear
迭代器

set和multiset的迭代器称为双向迭代器，不支持随机访问（不支持[]），支持星号*解除引用（访问值），支持++和–、==、!=等操作。

begin/end

–s.end()是指向集合中最大元素的迭代器。时间复杂度O(1)

insert

s.insert(x)将元素x插入到集合s中，时间复杂度O(logn)

在set中，若元素已经存在，则不会插入

find

s.find(x)会在集合中查找等于x的元素，并返回指向该元素的迭代器，若不存在，则返回s.end()。时间复杂度O(logn)。

lower_bound和upper_bound 二分

用法与find类似，时间复杂度O(logn)

s.lower_bound(x); 找到==大于等于==x的最小的元素的迭代器

s.upper_bound(x); 找到==大于==x的最小的元素的迭代器

erase

s.erase(it)从s中删除迭代器it指向的元素，时间复杂度O(logn)

s.erase(x)从s中删除所有等于x的元素，时间复杂度O(k+logn)，k为删除的个数

count

s.count(x) 返回集合中s等于x的元素的个数k，时间复杂度O(k+logn)，k为元素x的个数

注意，因为set不能存在重复元素，所以erase和count的k = 1 或 0

unordered_set 无序的set

#include <unordered_set>中包含unordered_set和unordered_multiset两个结构

unordered_set底层实现是哈希表，除了没有lower_bound和upper_bound函数，其他函数与set一致。

哈希表效率比较高，一般函数时间复杂度为O(1)，而set中是O(logn)

map

#include<map>包含map和multimap两个容器，

map容器是一个键值对的映射，其内部实现是一棵以key为关键码的红黑树。

map内部通过key构建红黑树，所以key必须定义小于号运算符。

常用方法

size/empty/clear/begin/end均与set类似
insert

与set一样，但是参数是pair<key_type, value_type>

同样，map中已存在k1时，插入相同的k1不同value的操作不会执行（不报错，原状态不变）。

erase

erase(key) 通过键值来删除

erase(iterator) 通过迭代器来删除

erase(iterator begin, interator end) 通过迭代器删除一个序列

erase的注意事项，在遍历中使用erase时，注意map是关联式容器

find

m.find(key) 通过key值来查找对应元素，返回的是元素的迭代器

[]操作符

m[key] 返回key映射的value的引用，时间复杂度为O(logn)

我们可以很方便地通过m[key]来得到key对应的value，还可以对m[key]进行赋值操作，改变key对应的value

m[key]++; 或 ++ m[key]操作，如果key之前不存在，会默认value值为0，然后进行相应操作。

for循环

unordered_map

同样是hash表，不可做二分，然后函数时间复杂度一般是O(1)

bitset

支持位运算的容器，用来存储很长的二进制数

pair

位运算

操作 符号 含义
与AND & 同1为1
或OR | 有1为1
取反NOT ~
异或XOR ^ 相同为1
左移 << 等价于乘$2^k$
右移 >> 等价于除$2^k$

两个整数进行位运算 == 对每一位分别进行位运算

常用操作

求x的第k位数字（从低位开始看） x >> k & 1

lowbit(x) = x & -x 返回x的最后一位的1

x & -x 等价于 x & (~x + 1)

双指针算法

第一类双指针算法

常用函数

reverse 翻转

可以翻转：数组、字符串、vector。时间复杂度：O(n)

注意第二项是最后一个元素的下一个位置！

unique 去重

进行去重操作，然后返回去重之后的尾迭代器/指针，这个迭代器是去重之后末尾元素的下一个位置。该函数常用于离散化，利用迭代器（或指针）的减法，可计算出去重后的元素个数。

注意：unique去重的前提条件是相同元素已经放到了一起，否则不相邻的重复数据不会被去除

把一个数组去重

vector、数组去重之后，都是将去重后的数据放到前半部分，但是后半部分没有用到的数据空间也不会删除，而是保留下来了。

random_shuffle

随机种子默认是0，一般设置时间为随机种子。

sort

不稳定排序，默认是从小到大排序

对结构体排序

lower_bound/upper_bound 二分

lower_bound(开始的迭代器/指针, 结束的迭代器/指针, x)

lower_bound的第三个参数传入一个元素x，在两个迭代器（指针）指定的部分上执行二分查找，返回指向第一个大于等于x的元素的位置的迭代器（指针）。

upper_bound与lower_bound基本一致，区别在于它查找的是第一个大于x的元素。

注意：这两个函数操作的对象都应该是提前排好序的！

next_permutation

next_permutation(开始的迭代器，结束的迭代器)

这个函数对传入的vector（或其他容器）的数据进行排序，返回下一种排序可能（默认按照数值从小到大的进行），返回值是是否为排序的最大情况，如果是返回true，否则返回false。（所以如果想获得所有的排序结果，需要先将vector按从小到大排列（即排列数值最小的情况））

取模运算

$$
(x * y) % m = ((x % m) * (y % m)) % m\=x%m*y%m
$$

$x%m*y%m$按照优先级，%优先级等于*，所以是从左到右，先取模再乘最后再取模。（注：这种写法不具有普遍性，需要保证x%m*y没有溢出，即y值相对较小时）
