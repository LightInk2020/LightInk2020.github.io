排序

快速排序

快速排序主要思想：**==分治==**
稳定排序 平均时间复杂度：$O(n\log n)$

步骤：

  1. 确定分界点
    1. q[l]
    2. q[(l + r)/2]
    3. q[r]
    4. 随机
  2. 调整区间
    1. 使得左边的数小于等于x
    2. 使得右边的数大于等于x
  3. 递归处理左右两端

调整区间的暴力做法：

  1. 开两个数组a[] b[]
  2. q[l~r]将其中
    1. q[i] ≤ x 数放到a[]
    2. q[i] ≥ x 数放到b[]
  3. a[] 放到 q[] b[]放到q[]
    ==调整区间的优美做法==：
    双指针，分别从左右两端一个指针，左边找不满足大于x的数，右边找不满足小于x的数，然后交换。

    不稳定 时间复杂度：$O(n\log n)$

    二分

    整数二分

二分的本质并不是单调性，单调的序列一定可以二分，但是不单调的序列也有可能可以二分。
二分的本质：只要区间可以找到一个性质使得左半部分和右半部分的性质不同，二分就可以实现。
二分模板一共有两个，分别适用不同情况。算法思路：假设目标值在闭区间[l, r]中，每次将区间长度缩小一半，当l = r时，就找到了目标值。

版本1

当我们将区间[l, r]划分为[l, mid]和[mid + 1, r]时，其更新操作是r = mid或l = mid + 1，计算mid时不需要加1。

版本2

当我们将区间[l, r]划分为[l, mid - 1]和[mid, r]时，其更新操作是r = mid - 1或者l = mid，为了防止死循环，计算mid时需要加1。

一维前缀和

#注意 前缀和下标一定要从1开始，s[0] = 0

二维前缀和

二维前缀和可以快速求出子矩阵的和。

以(x1, y1)为左上角，(x2, y2)为右下角的子矩阵的和为：
S[x2, y2] - S[x1 - 1, y2] - S[x2, y1 - 1] + S[x1 - 1, y1 - 1]

一维差分

构造b数组使得 a[i] = b[1] + b[2] + ... + b[i]，则b数组是a的差分，a是b的前缀和。
差分可以快速对a数组的[l, r]区间的数都加上常数c，即直接b[l] + c; b[r + 1] - c即可。时间复杂度O(1)。

更妙的写法

b[i]的构建相当于不断对一个小区间[i, i]加上一个值a[i]

二维差分

b[i][j]的构建相当于不断对一个小矩形左上角[i, j] 右下角[i, j]加上一个值a[i][j]

双指针

双指针可以使得时间复杂度从$O(n^2)$变为$O(n)$
思路：

  1. 暴力解法
  2. 找单调性。即i变化时，j的变化是否具有单调性。
  3. 优化

    (2) 对于两个序列，维护某种次序，比如归并排序中合并两个有序序列的操作
